# Dating site


## Installation
1. Open the terminal
2. Go into the directory you want to install the app in
```
git clone https://github.com/kylebot0/Package-BE.git
```
3. Install the dependecies 
```
npm install
```
4. Run the application
```
npm run start
```

## Usage 
To use the app run the application using 
```
npm run start
```
